# object

+ object是所有类的父类。一个类只能继承一个类，但object依然是所有类的父类，因为一个类不管怎么继承，它的父类的父类都是object，这是种巧妙的设计。
+ object仅有三种方法
+ object没有任何参数，因为它是最高的抽象。
